Как это работает?

Коротко

Алгоритм находит для каждого слова вашего предложения новое слово - самое на него похожее. Потом слово ставится в нужную форму (ту же, в которой было изначальное слово).

С подробностями

Самое похожее слово ищется с помощью библиотеки word2vec для python. Модель, на которой обучался алгоритм - web_mystem_skipgram_500_2_2015. Анализ и лемматизация словоформ производились с использованием библиотеки pymystem3 для python. Слова изменялись в нужную форму с помощью библиотеки pymorphy2 для python. Работа была вдохновлена игрой "Толстой или компьютер?".
