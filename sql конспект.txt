СУБД - система управления базами данных
Предметная область - информация, которую надо структурировать
Внешняя схема -> концептуальная схема -> внутренняя схема
Внешнее представление - совокупность требований к данным со стороны некоторой конкретной функции (реализуется с помощью концептуального проектирования)
Концептуальная схема - полная совокупность всех требований к даным (реализуется с помощью логического проектиирования)
Внутренняя схема - сама база данных (реализуется с помощью физического проектирования)
Функциональное моделирование: всякие диаграммы и прочее
